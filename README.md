Ghana news spider

# Setup

1. Install virtualenvwrapper and activate it: [install help](http://roundhere.net/journal/virtualenv-ubuntu-12-10/)

   ```
   $ mkvirtualenv news
   ```

   ```
   $ $VIRTUAL_ENV/bin/pip install -r requirement.txt
   ```


# Run a spider


  ```
  $ scrapy crawl <spider name>
  ```
